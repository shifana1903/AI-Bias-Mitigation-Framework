# Hybrid Framework for Bias Mitigation in AI

## 📌 Overview
This project presents a hybrid framework to detect and reduce bias in machine learning models.

## 🚀 Key Features
- Bias Detection using fairness metrics
- Causal Analysis to identify root causes
- Adversarial Learning for bias mitigation

## 📊 Results
Improves fairness with minimal accuracy loss.
